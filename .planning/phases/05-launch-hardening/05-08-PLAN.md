  <action>
    ```toml
    # Build config — keeps deploy behavior deterministic across reviewers.
    # SPA fallback stays in public/_redirects (already correct); no redirects block here.

    [build]
      command = "npm ci && npm run build"
      publish = "dist"

    [build.environment]
      NODE_VERSION = "22"
      # NPM_FLAGS intentionally unset — `npm ci` already enforces lockfile strictness.

    # Production-only headers: forbid inline eval + enforce HTTPS upgrade for any mixed-content.
    [[headers]]
      for = "/*"
      [headers.values]
        X-Frame-Options = "DENY"
        X-Content-Type-Options = "nosniff"
        Referrer-Policy = "strict-origin-when-cross-origin"
        Permissions-Policy = "geolocation=(), microphone=(), camera=()"
    ```

    Keep the headers block minimal — aggressive CSP deferred per 05-RESEARCH
    Security Domain (ASVS V14 partially covered; full CSP is a post-launch
    hardening item).
